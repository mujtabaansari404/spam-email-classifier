# Email Classifier Project

This project is a simple **Email Classifier** using Python. It classifies emails as spam or not spam based on the dataset provided.

## Dataset

The dataset for this project can be found here:  
[Email Dataset](https://www.kaggle.com/datasets/vjchoudhary7/customer-segmentation-tutorial-in-python)
